Range Slider

This is a simple, reusable, and customizable range slider web component. By clicking on the track of each sliders, users can modify the value of the slider and use the value to modify certain elements. By hovering over the thumb, users can view the current value of the slider.

Using the Component

To create a slider, use <range-slider id="mySlider"> </range-slider>.
By default, this will render the following slider, with step of 1. Users must specify an id for each component, so each sliders' value can be accessed via:
getSliderValue("<user specified id>")"

Attributes

This range slider component can have the following attributes. If an attribute is invalid, slider will be rendered with the default value.

Attribute Value Description
min number [pixels] The minimum value of the slider.
default value=0
max number [pixels] The minimum value of the slider.
Must be greater than min
default value=100
initial number [pixels] The initial value of the slider.
Cannot be less than min,
or greater than max.
default value= average of min and max
length number [pixels] The length of the track.
Must be greater than 0.
default value=200
height number [pixels] The height of the track.
Must be greater than 0.
This will also determine the height of the thumb.
default value=15
thumbRounding number [0 to 1] The roundness of the thumb.
Cannot be less than 0, or greater than 1.
default value=1
trackColor color [hex number] The color of the track.
Must be a color value in hex number with "#".
default value="#8FBC8F"
thumbColor color [hex number] The color of the thumb.
Must be a color value in hex number with "#".
default value="#FFFFFF"
thumbBorderColor color [hex number] The color of the thumb's border.
Must be a color value in hex number with "#".
default value="#8FBC8F"
showNumbers string "false" or "true" When set to true, the minimum and maximum values are displayed around the slider.
default value=true
showIndicator string "false" or "true" When set to true, the value of the slider is displayed when the pointer hovers over the thumb.
default value=true

Examples

<range-slider id="mySlider1"
trackColor="#e9c46a"
thumbColor="#f4a261"
thumbBorderColor="#90be6d"
> </range-slider>

<range-slider id="mySlider2"
initial = 100
min = -100
max = 100
length = 200
height = 10
thumbRounding = 0.3
trackColor = "#FFC0CB"
thumbBorderColor = "#008080"
> </range-slider>

<range-slider id="mySlider3"
min = 200
max = 10000000
length = 800
height = 10
> </range-slider>

<range-slider id="mySlider4"
showNumbers="false"
showIndicator="false"
> </range-slider>

Demo Codes

The first and the second demos are essentially equivalent.

getSliderTrack("mySliderDemo1").addEventListener('click', (event) => {
let sliderValue = getSliderValue("mySliderDemo1");
document.querySelector(".demo1").style.setProperty('opacity', sliderValue/100); });

If end user chooses to not use range-slider.js, please use the following example as reference.

document.getElementById("mySliderDemo2").shadow.querySelector("#mySliderDemo2"+"Track").addEventListener('click', (event) => {
let sliderValue = Number(document.querySelector("#mySliderDemo2").shadow.querySelector("#mySliderDemo2"+"Value").textContent);
document.querySelector(".demo2").style.setProperty('opacity', sliderValue/100); });
